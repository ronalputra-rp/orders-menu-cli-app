## How to Run

1. Make sure you have [Node.js](https://nodejs.org/) installed.  
2. Clone this repository or download the file `orders-menu-cli-app.js`.  
3. Open terminal/command prompt.  
4. Run the app using:

```bash
node orders-menu-cli-app.js
```
